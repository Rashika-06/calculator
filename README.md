# Advanced Calculator

A modern and responsive Calculator Application built using HTML, CSS, and JavaScript. This project was developed as part of my internship journey to strengthen my frontend development skills and gain hands-on experience with JavaScript logic and DOM manipulation.

## 🚀 Features

* Basic arithmetic operations (+, −, ×, ÷)
* Percentage calculations
* Bracket support for complex expressions
* Memory functions (MC, MR, M+, M−)
* Clear and delete functionality
* Responsive and user-friendly interface
* Clean and modern design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
